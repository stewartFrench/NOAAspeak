NOAAspeak is a continuous weather radio app that provides real-time NOAA forecasts and alerts with automatic text-to-speech broadcasting. 
Get weather updates spoken aloud continuously, just like traditional NOAA Weather Radio, for any US location.

NOAAspeak transforms your iPhone into a modern NOAA Weather Radio. 
The app automatically fetches official National Weather Service forecasts and speaks them aloud in a continuous loop, 
keeping you informed about current conditions, upcoming weather, and active alerts.

Key Features:

• Continuous Weather Broadcasting: Automatically speaks 6-period forecasts in a loop with 2-second pauses, mimicking traditional weather radio
• Active Weather Alerts: Immediate notification of severe weather warnings and watches with full details
• Location Flexibility: Use your current GPS location or search for any US location with autocomplete
• Voice Selection: Choose from multiple natural-sounding voices for weather narration
• Official NOAA Data: Direct access to National Weather Service forecasts and alerts via api.weather.gov
• Simple Controls: Listen/Stop button for continuous mode, easy location switching
• Clean Interface: Modern SwiftUI design with forecast cards showing temperatures, conditions, and wind information

Perfect for weather enthusiasts, outdoor workers, emergency preparedness, or anyone who wants hands-free weather updates while working, driving, or relaxing.

This app uses the official National Weather Service (NWS) API (api.weather.gov), which is publicly accessible government data provided by NOAA. According to NOAA's open data policy (https://www.noaa.gov/information-technology/open-data-dissemination), all NOAA data is in the public domain and freely available for any use. The API documentation is available at https://www.weather.gov/documentation/services-web-api and explicitly states the API is free to use for any purpose with no authentication required.
